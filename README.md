# Shahriar's Personal Portfolio Website

This is the repository for my personal portfolio website, built to showcase my projects, skills, and experience as a developer.

🌐 **Live Demo:** [https://Shahriar1638.github.io/MyPortfolio](https://Shahriar1638.github.io/MyPortfolio)

## 🛠️ Tech Stack

This project was built using modern full-stack web technologies:

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
- **Routing:** [React Router v7](https://reactrouter.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment:** GitHub Pages

## ✨ Features

- **Modern & Responsive UI:** Fully responsive design that works seamlessly across mobile, tablet, and desktop views.
- **Dynamic Animations:** Smooth page transitions and scroll animations powered by Framer Motion and AOS.
- **Project Showcase:** A dedicated section to display recent work and accomplishments.
- **Fast Performance:** Optimized build via Vite for lightning-fast load times.

## 🚀 Running Locally

If you'd like to explore the code or run the portfolio on your local machine, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation Steps

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Shahriar1638/MyPortfolio.git
   ```

2. **Navigate into the directory:**

   ```bash
   cd portfolio_frontend
   ```

3. **Install dependencies:**
   Using npm:

   ```bash
   npm install
   ```

   Or using bun (as `bun.lock` is present):

   ```bash
   bun install
   ```

4. **Start the development server:**

   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:5173` to see the localized version of the portfolio.
